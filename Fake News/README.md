## Fake News Prediction Modelling

### Developer Instructions

1) Create and activate environment
```bash
conda create -n fake_news python=3.10 -y

conda activate fake_news
```

2) Install dependencies
```bash
conda install numpy pandas scikit-learn py-xgboost matplotlib seaborn ipykernel wordcloud nltk textblob 
conda install -c conda-forge textstat
pip install scikeras[tensorflow]
```
