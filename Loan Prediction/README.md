## Loan Prediction Modelling

### Developer Instructions

1) Create and activate environment
```bash
conda create -n loan_predict python=3.10 -y

conda activate loan_predict
```

2) Install dependencies
```bash
conda install numpy pandas scikit-learn py-xgboost catboost matplotlib seaborn plotly ipykernel
```

### Development Cycle

1) Exploratory Data Analysis
2) Data Pre-processing
3) Training
4) Tuning & Feature Engineering
